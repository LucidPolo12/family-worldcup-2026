## The file (5 tabs)
- **How to Use** — quick reference.
- **Match Predictions** — all 104 fixtures pre-loaded (date, stage, group, teams). Each player has Pred H / Pred A / Pts columns. You type predicted scores and the actual final score in the two ACTUAL columns; points auto-calculate. Knockout rows are shaded.
- **Schedule** — full fixture list with venues and kickoff times (ET). Use for "make your picks" posts.
- **Tournament Picks** — each player's Champion / Runner-Up / 4 Semifinalists, plus an ACTUAL RESULTS block to fill as stages are reached.
- **Leaderboard** — fully automatic: totals, rank, exact-score count, accuracy, champion picks.

## Scoring — matches
- Correct result (right winner, or correctly calling a draw) = **3 points**.
- Exact score = **8 points** (3 + 5 bonus). Anything else = **0**.
- A correct draw works the same way: predicting 1–1 when it ends 2–2 = 3; exact 1–1 = 8.
- **Group stage:** players predict as many or as few matches as they like.
- **Knockout:** predictions are the score at the end of regulation (90 + stoppage). Extra time and penalties do not count — a 1–1 that goes to penalties scores as 1–1.

## Scoring — pre-tournament (locked before the opening match)
- Champion = 25, Runner-Up = 15, each correct Semifinalist = 10 (four picks, max 40). **Maximum 80.**

## Tie-breakers (in order)
Most exact scores → most correct outcomes → correct champion pick → shared.

## Conventions
- Picks are written **home team first**: "Mexico 3–0 South Africa" = home 3, away 0. If a pick is phrased "Team X to win 2–1," confirm which side is home before entering.
- Points only count once the actual final score is entered.
- "Champion still alive" is a manual call based on eliminations — ask me if unsure.
- Flag anything ambiguous instead of guessing.
